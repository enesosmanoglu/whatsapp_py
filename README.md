## Installation
```bash
pip install https://github.com/enesosmanoglu/whatsapp_py/archive/main.zip
```

## Features
- [x] Send text messages
- [x] Send files
- [x] Send media
- [x] Send messages at a specific time
- [x] Send delayed messages
